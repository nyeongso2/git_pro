# ✨ Antigravity Diary & Planner

> "당신의 소중한 하루를 기록하고, AI와 함께 돌아보는 프리미엄 다이어리 서비스"

![Diary App Banner](https://images.unsplash.com/photo-1517842645767-c639042777db?auto=format&fit=crop&q=80&w=1000)

## 🌟 주요 기능 (Key Features)

### 1. 스마트 다이어리 & 기분 트래킹
- **감성 기록**: 그날의 분위기를 반영하는 6가지 감정 이모티콘과 함께 일상을 자유롭게 기록하세요.
- **프리미엄 에디터**: 깔끔한 디자인의 에디터에서 방해받지 않고 글을 작성할 수 있습니다.

### 2. AI 기록 요약 & 히스토리
- **AI 한 줄 요약**: 작성한 긴 글을 AI가 분석하여 사이드바에 핵심 내용을 요약해줍니다.
- **멀티 기록 관리**: 한 날짜에 여러 번 기록할 수 있으며, 과거 기록을 리스트로 확인하고 즉시 불러와 수정할 수 있습니다.
- **개별 삭제**: 불필요한 이전 기록은 리스트에서 바로 삭제 가능합니다.

### 3. 시간 기반 할 일 목록 (Planner)
- **시간 설정**: 각 할 일마다 수행 시간을 설정할 수 있습니다.
- **자동 정렬**: 추가한 할 일들은 설정된 시간에 맞춰 자동으로 순서가 정렬됩니다.
- **체크리스트**: 완료된 항목은 직관적인 애니메이션과 함께 표시됩니다.

### 4. 스마트 캘린더 & 기념일
- **직관적 탐색**: 사이드바 캘린더를 통해 과거와 미래의 일정을 한눈에 파악하세요.
- **기념일 마킹**: 소중한 날을 기념일로 설정하면 달력에 예쁜 별(⭐) 표시와 함께 강조됩니다.

### 5. 보안 및 데이터 관리
- **로그인 시스템**: 나만의 계정으로 접속하여 개인적인 기록을 안전하게 보호하세요.
- **게스트 계정**: 별도의 가입 없이 `guest123` / `guest123`으로 즉시 체험 가능합니다.
- **로컬 저장소**: 모든 데이터는 브라우저의 로컬 저장소에 안전하게 유지됩니다.

## 🎨 기술 스택 (Tech Stack)

- **Frontend**: React, TypeScript, Vite
- **Styling**: Vanilla CSS (Custom Design System), TailwindCSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Date Handling**: Date-fns

## 🚀 시작하기 (Getting Started)

### 사전 준비
- Node.js (v16 이상 권장)
- npm 또는 yarn

### 설치 및 실행
```bash
# 1. 의존성 설치
npm install

# 2. 로컬 서버 실행
npm run dev
```
